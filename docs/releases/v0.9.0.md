<h1 align="center" style="margin-bottom: 4px;">Topo v0.9.0 🦔</h1>
<p align="center" style="margin-top: 4px;"><em>Visual Unity & Deep Package Logic.</em></p>

Topo v0.9.0 is a comprehensive update focused on aesthetic consistency, low-level system optimization, and a rigorous white-box security audit of the deletion core. This release marks the transition to a fully unified branded theme and introduces intelligent package dependency management.

## Highlights

### Unified Branded Aesthetics

- **Centralized Theme Engine**: Introduced `THEME_TITLE` as a global constant, ensuring identical Bright Purple branding across all interactive selectors and CLI headers.
- **Icon Modernization**: Upgraded the standard folder emoji `📁` to the more professional and tactile card index dividers `🗂️` throughout the analyzer and documentation.
- **Viewport Balance**: Refined TUI headers with consistent padding and spacing, providing a more professional and centered visual experience.
- **Navigation Mute Toggle**: Users can now press **`M`** in the main menu to toggle navigation sounds on or off.

### Advanced System Cleanup

- **Orphaned Package Removal**: Integrated cross-distro dependency cleanup. Topo now detects and removes orphaned packages using `dnf autoremove` (Fedora), `apt-get autoremove` (Ubuntu), and `pacman -Rns` (Arch).
- **Zombie Process Reaping**: Automatically identifies processes in the `Z` (defunct) state and signals their parents via `SIGCHLD` to safely reclaim system PID resources.
- **Smart Workspace Protection**: Hardened the uninstaller to ignore visible top-level home directories (like `~/Projects` or `~/Code`), focusing residue searches exclusively on hidden dot-directories.
- **Project Purge Refinement**: The `bin/` directory is now only treated as a build artifact when accompanied by .NET project files, preventing accidental deletion of script folders.

### Security Audit & Deletion Hardening

- **Recoverable Residue Removal**: Heuristic application residue removal now defaults to **Trash** (`use_trash=True`), ensuring that any automated mismatch is non-destructive and easily recoverable.
- **Atomic Path Resolution**: Fixed a race condition where privileged deletions could validate a symlink but execute on a different path; Topo now resolves targets once and executes strictly on the verified absolute path.
- **Symlink Amplification Guard**: Proactive app detection now explicitly skips symlinks to prevent cleanup tasks from traversing into out-of-tree directories.
- **Installer Payload Validation**: The self-update mechanism now verifies the integrity of the downloaded installer script (shebang check) before execution.

### CLI Modernization & UX

- **Help Layout Overhaul**: Reworked `topo --help` into a modern `Quick Start / Whitelist / Notes` structure with real-world usage examples and command-specific routing.
- **Strict Sudo Interactivity**: Refined the sudo pre-prompt to accept only `Enter`, `Space`, and `ESC`, while safely consuming and ignoring other terminal escape sequences.
- **HTTPS Status Lookup**: Switched public IP diagnostics to use secure HTTPS endpoints (`https://ipinfo.io`) by default.
- **Contextual Dry-Run**: Optimized command routing so preview flags live only on relevant subcommands (e.g., `topo clean --dry-run`).

## Reliability & Performance

- **Rust Engine Safety**: Implemented `saturating_add` for all file size and count accumulations to ensure overflow safety on massive storage volumes.
- **Test Suite Robustness**: Added precise file stat mocking for Python 3.14+ environments and isolated container-specific tests for cross-platform stability.


## Notes

- Existing manual whitelist rules in `~/.config/topo/whitelist.json` are preserved.
- The `topo history` command now supports escaping for control characters in audit logs.
- System cleanup tasks now accurately account for reclaimed space in multi-stage package manager runs.

## 中文重点

Topo v0.9.0 专注于视觉风格的深度统一、底层系统组件优化以及核心删除引擎的白盒安全性审计与加固。

### 重点更新

- **品牌视觉统一**：引入全局 `THEME_TITLE` 常量，将全线 TUI 视图和 CLI 标题统一为品牌紫；全项目文件夹图标升级为更精致的 `🗂️` 索引卡。
- **深度包管理优化**：新增孤儿包自动清理（支持 DNF, APT, Pacman）以及僵尸进程安全回收机制。
- **安全加固（审计修复）**：修复了卸载逻辑中对顶级可见目录（如项目文件夹）的误匹配风险；卸载残留清理现在默认移入回收站而非直接删除。
- **符号链接保护**：加固了主动探测逻辑，防止清理任务通过软链接误入非目标目录；优化了 `sudo rm -rf` 的路径解析一致性。
- **帮助信息现代化**：重构了 `topo --help` 布局，采用分层结构展示快捷示例、白名单策略和安全说明。

### 技术改进

- **Rust 引擎防溢出**：底层扫描引擎引入饱和加法，确保在处理海量小文件或超大容量磁盘时数据不溢出。
- **跨平台兼容性**：针对 Python 3.14+ 和不同发行版的 `stat` 差异优化了测试套件，目前 186 项测试在各主流环境下已 100% 通过。
