# Topo v0.9.6

Topo v0.9.6 introduces significant performance optimizations for disk analysis and application uninstallation, enhances terminal interaction with native scrolling and mouse support, and strengthens security with GPG verification for updates.

## Highlights

### High-Performance Disk Analysis
- **Responsive "Preview" Mode**: Directories with more than 500 direct children (e.g., large caches or `node_modules`) now trigger a "Fast Explore" mode. This mode samples the first 500 entries instantly without recursive size calculation, keeping the UI responsive.
- **Improved Rust Integration**: Expanded the use of the Rust engine (`get_size_fast()`) across the entire project, including Trash cleanup, Analyze Disk, and uninstallation previews, resulting in a near-instant experience for large directory trees.
- **Redundancy Cleanup**: Consolidated size parsing and removed recursive Python fallback logic in critical paths to ensure the fastest possible metadata gathering.

### Optimized Uninstall Workflow
- **Instant Scan Feedback**: Entering the Uninstall menu now immediately displays a "Scanning installed applications..." message, eliminating the feeling of a "stuck" UI while package managers are queried.
- **Short-Term Scan Cache**: Implemented a 30-second TTL scan cache for installed applications. Re-entering the Uninstall menu from the Main Menu is now instantaneous.
- **Smart Cache Invalidation**: The scan cache uses a composite key (OS ID + desktop folder signature) and is automatically cleared after a successful uninstallation to ensure data accuracy.

### Enhanced Terminal UX
- **CLI-First Scrolling**: Result output from Clean, Optimize, and Status now uses the terminal's native scrollback. You can now use your standard terminal scrollbar or mouse wheel to review long cleanup logs.
- **Mouse Wheel Selection**: Restored and refined mouse wheel behavior in TUI lists. Scrolling now moves the selection arrow predictably instead of scrolling the whole frame.
- **Lightweight Visuals**: Introduced a minimal gray scrollbar and added a `show_scrollbar` configuration option.
- **Main Menu Polish**: Added mouse support to the main menu and ensured consistent screen clearing when launching commands.

### Security & Reliability
- **Package Update GPG Verification**: `topo update` now automatically verifies the checksum manifest against the Topo release public key when GPG is available, ensuring the authenticity of downloaded packages.
- **Hardened Downloads**: The updater now uses atomic `.part` files and retries (up to 4 times) to handle intermittent network failures during package downloads.
- **Shared Heavy Cache Metadata**: Centralized metadata for system caches (Apt, Dnf, Pacman), containers (Docker, Podman), and AI models (HuggingFace, Ollama) to ensure consistency between Analysis and Cleanup.

# Topo v0.9.6 中文重点

Topo v0.9.6 带来了显著的性能飞跃和交互改进，主要集中在磁盘分析、应用卸载以及终端原生体验上。

### 核心改进

- **极速磁盘分析**：针对包含超大量文件的目录（如浏览器缓存、`node_modules`）引入了“预览模式”。当目录直接子项超过 500 个时，会跳过递归统计实现秒级开启，确保 UI 响应如丝般顺滑。
- **卸载流程优化**：
    - **秒开体验**：引入 30 秒短期扫描缓存，短时间内再次进入卸载列表不再需要漫长的包管理器查询。
    - **即时反馈**：进入扫描时立刻提示状态，消除空白等待感。
    - **自动刷新**：卸载成功后自动失效缓存，确保应用列表实时准确。
- **终端交互升级**：
    - **原生滚动**：清理、优化和状态输出现在使用终端自带的滚动条，方便用户查阅长日志，不再受限于固定高度的框架。
    - **鼠标增强**：主菜单全面支持鼠标点击；列表视图支持鼠标滚轮精准控制高亮选择。
    - **轻量化视觉**：更换了更简洁的滚动条样式，并支持在配置中完全隐藏。
- **安全性与可靠性**：
    - **GPG 签名验证**：`topo update` 现在支持验证发布包的 GPG 签名，确保您的更新来自官方且未被篡改。
    - **下载鲁棒性**：引入了断点续传重试机制，增强了网络不稳定环境下的更新成功率。
- **底层重构**：全面集成 Rust 引擎进行极速大小计算，移除了大量冗余的 Python 递归逻辑，系统整体响应显著加快。
