<h1 align="center" style="margin-bottom: 4px;">Topo v0.8.0 🐘️</h1>
<p align="center" style="margin-top: 4px;"><em>Tactile Interaction & Security Hardening.</em></p>

Topo v0.8.0 is a massive update focused on TUI responsiveness, high-fidelity interaction, and a deep security audit to ensure your system and data remain safe while digging. This release transforms the interaction model with native mouse wheel support and real-time terminal resizing.

## Highlights

### Responsive & Interactive TUI

- **Terminal Resize Handling**: Implemented dynamic size polling. The scrollbar now stays perfectly attached to the right edge during window dragging, and artifacts are automatically cleared when shrinking the terminal.
- **Mouse Wheel Support**: Added native vertical wheel scrolling (3 lines per notch) across all list views (Analyze, Uninstall, Clean).
- **In-TUI Deletion Confirmation**: Integrated the two-step confirmation prompt directly into the Analyze interface. Count and size details now appear as a status line below selected items, maintaining UI focus.
- **Copy-Paste Compatibility**: Selective mouse tracking allows instant text selection in the Main Menu and Confirm dialogs without holding **Shift**.

### Elite System Optimization

- **Smart Swap Management**: Automatically resets Swap only when physical RAM is plentiful (available RAM > 2x used swap), migrating data back to memory to eliminate micro-stutters.
- **Aggressive Journal Maintenance**: Added a task to vacuum systemd journals to a compact 3-day history.
- **System Coredump Cleanup**: Reclaim gigabytes by clearing historical application crash dumps (`/var/lib/systemd/coredump`).
- **Snap Cache Cleanup**: Proactively cleans user caches for Snap applications (located in `~/snap/*/common/.cache`) previously unmanaged by generic cleaners.

### Security & Safety Pass (Audit Fixed)

- **Critical User-Data Protection**: Hardened the uninstaller to never treat standard XDG folders (`Music`, `Videos`, `Pictures`, etc.) as app residue, ensuring your media libraries are safe from accidental cleanup.
- **Reliable Process Termination**: Rewrote the application "Kill" logic to use real executable names derived from `.desktop` files, ensuring apps are properly closed before removal.
- **Symlink-Safe Maintenance**: Updated age-based cleanup to use `lstat`, preventing it from following symlinks or aborting on broken ones.
- **SQLite Integrity**: Added guaranteed connection closing for browser database optimizations to prevent file-handle leaks.

### Professional Auditory Feedback

- **Bundled Sound Assets**: Introduced high-quality `cli_click.wav` and `delete_remove.wav` for tactile confirmation of navigation and destructive actions.
- **Intelligent Playback**: Asynchronous, low-latency playback using `pw-play`, `paplay`, or `aplay`. It falls back gracefully to the terminal bell if audio infrastructure is missing.
- **Installation Preservation**: Updated `install.sh` to correctly preserve the `assets/` directory and WAV files during deployment to `~/.topo`.

### Robust Lifecycle Management

- **Safer Self-Update**: `topo update` now uses a shell-free, metacharacter-safe update flow with explicit network timeouts and semantic version validation.
- **Thorough Removal**: `topo remove` now cleans up its own audit logs and restores shell PATH configurations by stripping the `# Added by topo` block.
- **Atomic Linking**: Command symlinks are now created atomically to prevent corrupted installations during manual link updates.


## Notes

- Mouse wheel support requires a terminal that supports SGR or X11 mouse protocols (enabled by default in GNOME Terminal, Konsole, Alacritty, Kitty, and Windows Terminal).
- Auditory feedback can be customized by placing your own `click.wav` or `delete.wav` in `~/.config/topo/sounds/`.
- Deletion logs continue to be available at `~/.local/state/topo/deletions.log` for auditing.

Topo v0.8.0 是一个里程碑版本，带来了更灵动的交互响应、更专业的系统维护能力以及全方位的安全性加固。

### 重点更新

- **响应式交互**：实现了终端窗口缩放自适应，滚动条实时贴合边缘；全面支持鼠标滚轮滚动和界面内集成删除确认提示。
- **深度优化增强**：新增智能 Swap 重置（消除由于磁盘交换引起的系统微卡顿）、3天级激进日志维护、系统核心转储文件 (Coredump) 清理以及 Snap 应用缓存深度清理。
- **安全加固 (审计修复)**：紧急修复了卸载逻辑中对 XDG 标准用户目录（音乐、视频等）的误匹配；加固了对软链接的处理，防止清理流程中意外中断或误跳出。
- **高保真音效**：内置专属于 Topo 的导航点击声与操作确认音，提供即时、有质感的听觉反馈，并支持用户本地音效覆盖。
- **更可靠的生命周期**：优化了自更新脚本的安全性，防止 Shell 注入；完善了卸载流程，自动清理环境变量及审计日志。
