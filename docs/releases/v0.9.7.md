# Topo v0.9.7

Topo v0.9.7 focuses on architectural refinement and diagnostic capabilities. It introduces the `topo doctor` command, a modernized TUI with a compact vector-style banner, robust terminal interrupt recovery, and hardened system optimization tasks.

## Highlights

### Diagnostic & Robustness
- **New `topo doctor` Command**: A lightweight diagnostic tool that probes OS architecture, Rust engine integrity, package manager availability, and configuration paths to troubleshoot installation issues instantly.
- **Terminal Interrupt Recovery**: Implemented a global terminal state registry and signal handlers (SIGINT/SIGTERM). Topo now gracefully restores your terminal (cursor, raw mode, alternate screen) even if interrupted during a password prompt or a scan.
- **Flicker-Free TUI Refactoring**: Transitioned the main menu to a string-based rendering model and action-based routing. This eliminates standard output redirection and provides a more stable, extensible internal architecture.

### Compact Modern Branding
- **Braille Vector Banner**: Replaced the large block ASCII banner with a sophisticated 2-line "vector" mascot using Braille patterns. This frees up significant vertical space while maintaining a high-end, modern terminal aesthetic.
- **Single Version Source**: Centralized versioning into `src/core/constants.py`, ensuring the version displayed in the banner and CLI is always synchronized with the root `VERSION` file.

### Privacy & Offline Hardening
- **Zero-Network IP Detection**: Reworked local IP detection to use kernel routing table analysis (`/proc/net/route`) and direct `SIOCGIFADDR` ioctl calls. Topo no longer requires an external connection (e.g., 8.8.8.8) to identify your local network address.
- **Public IP Removed**: Completely removed the public IP lookup feature to maximize user privacy and ensure the tool remains fully functional in isolated or firewalled environments.

### Hardened Optimization Tasks
- **Robust Coredump Cleanup**: Switched to a physical removal fallback using `find -delete` for `/var/lib/systemd/coredump/`, ensuring large crash files are reclaimed even when journal tools are insufficient.
- **Precise Autostart Management**: Integrated the `shlex` library to correctly parse complex `Exec=` lines in `.desktop` files, preventing false-positive removals of valid startup items with spaces or quotes.
- **GPU Shader Cache Pruning**: Added age-based cleanup (7-day threshold) for NVIDIA and Mesa (AMD/Intel) shader caches to reclaim space from stale graphics data without impacting active performance.

# Topo v0.9.7 中文重点

Topo v0.9.7 侧重于架构层面的深度重构和诊断能力的提升，旨在为 v1.0.0 稳定版奠定坚实的基础。

### 核心改进

- **全新 `topo doctor` 命令**：新增轻量化诊断工具，可一键排查操作系统环境、Rust 引擎健康度、包管理器依赖及权限状态，极大方便了安装问题的定位。
- **极致的终端恢复力**：引入全局终端状态管理和信号监听。现在，即使在输入 sudo 密码或进行大规模扫描时按下 `Ctrl+C`，Topo 也能瞬间恢复光标、原始模式和屏幕缓冲区，绝不弄乱你的终端。
- **现代化极简 Banner**：将原本臃肿的 5 行 ASCII Banner 替换为仅占 2 行高度的“Braille 矢量点阵”图标。在保留品牌辨识度的同时，为核心功能区腾出了更多视野。
- **纯离线隐私加固**：
    - **内核级 IP 探测**：通过直接解析内核路由表和 ioctl 调用获取本地 IP，不再依赖任何外部网络连接。
    - **移除公网 IP 查询**：彻底删除公网 IP 探测功能，保证工具在内网或防火墙环境下依然极速响应，保护用户隐私。
- **加固的清理任务**：
    - **物理清理 Coredump**：针对系统崩溃文件引入物理删除回退机制，清理更彻底。
    - **精确识别自启动项**：使用 `shlex` 解析复杂的自启动配置文件，确保带空格或引号的路径能被安全识别。
    - **GPU 缓存老化清理**：新增对 NVIDIA 和 Mesa 显卡着色器缓存的 7 天阈值自动清理。
- **底层架构升级**：统一了全局版本号来源，并将 TUI 路由重构为 Action ID 模式，代码更健壮、易于扩展。
