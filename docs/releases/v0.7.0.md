<h1 align="center" style="margin-bottom: 4px;">Topo v0.7.0 🐿️</h1>
<p align="center" style="margin-top: 4px;"><em>Deep clean and optimize your Linux.</em></p>

Topo v0.7.0 focuses on performance optimizations for disk analysis, a more thorough uninstallation process for Debian/Ubuntu systems, and a unified "Mole-style" UI experience across the entire application.

This release significantly improves the speed of the disk analyzer and ensures that uninstalled applications leave even less of a footprint on your system.

## Highlights

### High-performance Disk Analysis

- Added a full-subtree cache to the disk analyzer. Previously, drilling into a directory would trigger a rescan; now, a single pass from the Rust engine aggregates data for all subdirectory levels.
- Updated the Rust `topo-core` engine with a new `--tree` mode for hierarchical data aggregation.
- Reduced UI jitter by adding a 0.15s grace period to the scan spinner—instant cache hits now display results immediately without flickering.
- Moved the analysis scan status to a standalone page for better visibility.

### Thinner Uninstallation Footprint

- **APT (Debian/Ubuntu) Enhancement**: Changed the uninstallation command to `apt purge -y --autoremove`. This ensures that system-wide configuration files and unused dependencies are removed along with the application.
- Refined residue discovery logic to be more precise while maintaining safety.
- Clarified the context of "App Data Removal" in confirmation screens to better inform users about what is being deleted.
- Updated the uninstaller UI to use "Remove" terminology consistently and improved its visual layout.

### Unified "Mole-style" UI/UX

- Refactored all interactive menus and confirmation screens to follow a consistent, sleek aesthetic inspired by modern CLI tools.
- Unified the user experience for all destructive actions (Clean, Purge, Uninstall) with standardized confirmation prompts and shortcuts.
- Improved the readability of system metrics in `topo status`, particularly CPU load averages.
- Added support for canceling `sudo` password prompts using the **Escape** key.
- Aligned selection shortcuts across all paginated lists: `Space` for selection, `Enter` for confirmation, and `A` for selecting all.

### Hardened System Protection

- Updated the default whitelist to include core GNOME components and other critical system paths.
- Hardened the protected path policy to prevent accidental deletion of sensitive system configuration even in "Deep" mode.
- Improved the robustness of Linux-specific optimization tasks like `fstrim` and system database maintenance.

### Reliable Installation & Updates

- The `install.sh` script now defaults to installing the latest stable release from GitHub rather than the development branch.
- Added support for "Quiet" installs when specifying a specific version tag.
- Automatic trimming of unnecessary files during the installation process to keep the project footprint small.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/Jesencloud/Topo/main/install.sh | bash
```

## Verified environments

- Fedora 40 (Workstation Edition)
- Ubuntu 24.04 (LTS)
- Arch Linux (Rolling)

## Notes

- The new disk analysis cache significantly reduces disk I/O when exploring large file trees.
- For Ubuntu/Debian users, the new `purge` behavior means that system-level config files (usually in `/etc`) will now be removed. User-level configs in `~` continue to be handled by Topo's residue cleaner.
- Deletion logs continue to be available at `~/.local/state/topo/deletions.log` for auditing.

## 中文重点

Topo v0.7.0 带来了显著的磁盘分析性能提升、更彻底的 Ubuntu/Debian 卸载逻辑，以及全平台统一的 “Mole 风格” 交互体验。

### 重点更新

- **极速磁盘分析**：引入全树缓存机制。现在 Rust 引擎单次扫描即可获取所有层级的数据，进入子目录不再需要重新扫描，实现“秒开”体验。
- **更彻底的卸载 (APT)**：针对 Ubuntu/Debian 系统，将卸载命令升级为 `apt purge --autoremove`。除了应用本身，还会自动清理系统级配置文件和不再需要的孤立依赖包。
- **统一交互视觉**：全线重构 TUI 界面，采用统一的紫色/青色调视觉风格。规范了所有破坏性操作（清理、卸载、彻底删库）的确认逻辑。
- **操作逻辑优化**：支持使用 `ESC` 键取消 `sudo` 授权提示；对齐了所有列表的选择快捷键（空格选择、回车确认、A 全选）。
- **更强的路径保护**：进一步加固了白名单策略，默认保护 GNOME 核心组件及关键系统配置，防止在深度清理时误删。
- **安装程序优化**：`install.sh` 默认改为安装 GitHub 上的最新稳定版，并支持静默安装特定版本。

curl -fsSL https://raw.githubusercontent.com/Jesencloud/Topo/main/install.sh | bash
